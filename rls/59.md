# `0x59` Light Sensor Status

RLS `0xe8` â†’ LCM `0xd0`.

This command reports the status and demand for the light state to the LCM.

## Examples

    E8 05 D0 59 50 00 34
    E8 05 D0 59 32 02 54
    E8 05 D0 59 11 01 74

## Properties

Fixed length. Two byte bitfield.

    # Byte 1

    INTENSITY       = 0b0111_0000
    LIGHT_STATE     = 0b0000_0001
    
    # Byte 2

    REASON          = 0b0001_1111


#### Intensity  `0b0111_0000`

    INTENSITY_1  = 0x10
    INTENSITY_2  = 0x20
    INTENSITY_3  = 0x30
    INTENSITY_4  = 0x40
    INTENSITY_5  = 0x50
    INTENSITY_6  = 0x60

#### Light State `0b0000_0001`

    LIGHTS_ON  = 0x01
    LIGHTS_OFF = 0x00

---

#### Reason `0b0001_1111`

    TWILIGHT    = 0x01
    DARKNESS    = 0x02
    RAIN        = 0x04
    TUNNEL      = 0x08
    GARAGE      = 0x10
