## Parameters

Fixed length. 2 byte bit field.

The function of byte 2 is unknown, probably unused.

    # Byte 1

    SOURCE       = 0b0000_0001
---



### Source `0b0000_0001`

    SOURCE_TV    = 0b0001  # 0x01
    SOURCE_RAD   = 0b0000  # 0x00

---

## Use Cases

The command is used to switch the input mode of the radio module.

It is sent by the GT when switching to TV mode and back.
